# 台球搭子 (Billiard Buddy)

这是一个基于 Flutter 开发的移动应用程序，旨在为台球爱好者提供一个寻找球友、参加比赛以及管理台球活动的平台。

## 🚀 项目简介

“台球搭子” 提供了完整的约球闭环体验：从发布约球信息、成员申请加入、发起人审批，到最后的现场扫码核销。

## 💡 技术难点与亮点

### 1. 列表滑动性能优化 (Scrolling Performance)
在首页“动态广场”中，为了确保在包含大量高清图片的情况下依然保持 60fps 的流畅滑动体验，项目实施了以下优化：
- **图片加载策略**: 采用 `frameBuilder` 实现图片的渐进式淡入效果，减少图片加载时的视觉抖动。
- **预占位机制**: 为头像和内容图片设计了固定比例的 Loading 占位符，有效防止了图片加载完成后的“布局抖动 (Layout Shift)”。
- **异步分块处理**: 在进入页面前对扁平化的 API 数据进行预处理（按 `postid` 分组并合并图片列表），避免了在列表构建过程中进行复杂的计算逻辑。

### 2. 复杂数据处理与兼容
- **健壮的时间解析**: 针对后端返回的多种非标准时间格式（包含反引号、不同分隔符等），编写了专门的 `_tryParsePostTime` 解析器，确保动态排序的准确性。
- **数据清洗层**: 在 Model 层对后端返回的脏数据（如带反引号的字符串）进行统一清洗，保证了 UI 展示的整洁。

### 3. 核销闭环系统
- **跨端权限处理**: 统一封装了 iOS/Android 的摄像头权限申请逻辑。
- **状态同步**: 实现了“申请 -> 同意/拒绝 -> 待支付 -> 已支付 -> 现场扫码核销”的完整状态机切换逻辑，并实时同步 UI 状态。

## ✨ 核心功能

- **约球广场**：浏览附近的约球动态和场馆推荐。
- **活动发布**：支持发布不同类型的台球活动（如：斯诺克、中式八球等），设置费用模式和球技要求。
- **成员管理**：发布者可以查看申请列表，进行“同意”或“拒绝”操作。
- **核销系统**：
  - **二维码生成**：用户到场后可出示核销二维码。
  - **扫一扫**：商家或发布者通过扫码完成活动核销，触发保证金退还等逻辑。
- **个人中心**：管理“我的发布”、“我的参与”，支持个人资料编辑。
- **身份验证**：完整的登录、注册及找回密码流程。

## 🛠️ 技术栈

- **框架**: [Flutter](https://flutter.dev/) (Dart)
- **网络请求**: `http`
- **数据持久化**: `shared_preferences`
- **扫码插件**: `mobile_scanner`
- **二维码生成**: `qr_flutter`
- **模型序列化**: `json_annotation`, `json_serializable`
- **资源管理**: 自定义字体 (AlibabaPuHuiTi) 和多分辨率图片素材

```text
├── pages/        # 各功能模块页面

│   ├── Auth/     # 登录注册相关
│   ├── Main/     # 首页、动态
│   ├── Message/  # 消息中心
│   ├── Mine/     # 个人中心及核销组件
│   └── VenueList/# 场馆与赛事详情
├── routes/       # 路由配置
├── utils/        # 工具类 (存储、网络、颜色扩展)
└── widgets/      # 通用 UI 组件
```
